Quick Reference

API Cheat Sheet

HTTP endpoints your Python code calls under the hood. Works from any browser.

💡 LED Control
  • GET /l1on — Turn LED on
  • GET /l1off — Turn LED off
  • GET /toggle — Toggle LED
⚙️ Servo Control
  • GET /servo?num=1&angle=90 — Move servo
  • GET /s1on / GET /s1off — Quick action
  • GET /s1_120 — Set angle directly
🌈 NeoPixel
  • GET /neopixel?pixel=0&r=255&g=0&b=0
  • GET /neopixel_all?r=0&g=255&b=0
  • GET /neopixel_clear
🕒 Scheduling
  • GET /schedule?action=toggle&delay=5
  • GET /schedule?action=l1on&at=1714000000
  • GET /status / GET /time
