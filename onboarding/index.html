Lesson 1

Output is Evidence

Goal: Understand that print() sends visible output and that quotes mean text.

📖 Key Ideas

  • Code is instructions — the computer follows them exactly.
  • Python runs top to bottom, one line at a time.
  • print() shows a value on screen — it's how you see what your code does.
  • Text must go inside quotes — that's how Python knows it's text, not a command.

What just happened?

When you clicked "Wave", your browser sent a tiny message across your Wi-Fi network. That message traveled at the speed of light — 186,000 miles per second — to your robot's brain, an ESP32 microchip no bigger than a postage stamp.
